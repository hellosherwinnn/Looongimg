# LooongImg Standalone Web App 📸

A powerful, fully decentralized web application for stitching long scrolling screenshots using **FFmpeg.wasm**. 

一个基于 **FFmpeg.wasm** 的全去中心化长截图拼接 Web 应用。

## ✨ Features / 核心特性
- **Pure Client-Side / 纯客户端计算**: No uploads, 100% private. / 视频无需上传，保证100%隐私安全。
- **Zero Cost / 零成本运营**: Can be hosted for free on GitHub Pages. / 可作为纯静态文件托管在 GitHub Pages 等平台，零服务器运行成本。
- **Cross-Platform / 跨平台兼容**: Works in any modern browser with WebAssembly support. / 支持任何现代浏览器（依靠 WebAssembly），手机/电脑双平台互通。

## 🚀 Getting Started / 快速开始

### Prerequisites / 环境需求
- Node.js (v18+)

### Installation / 安装步骤
```bash
# In the standalone-web directory / 在 standalone-web 目录下运行：
npm install
```

### Running Locally / 本地运行
```bash
npm run dev
```

### Deployment (GitHub Pages) / 一键部署至 GitHub Pages
```bash
npm run deploy
```
