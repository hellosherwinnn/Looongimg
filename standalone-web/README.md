# LooongImg Standalone Web App 📸

[English] | [中文](./README_CN.md)

A powerful, fully decentralized web application for stitching long scrolling screenshots using **FFmpeg.wasm**.

## ✨ Features
- **Pure Client-Side**: No uploads, 100% private. All video processing happens in your browser.
- **Zero Cost**: Can be hosted for free on GitHub Pages.
- **Full Compatibility**: Optimized for both **Desktop** and **Mobile** (iOS/Android) browsers.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)

### Installation
```bash
# In the standalone-web directory:
npm install
```

### Running Locally
```bash
npm run dev
```

### Deployment (GitHub Pages)
```bash
npm run deploy
```
