# LooongImg 纯前端脱机版 📸

[English](./README.md) | [中文]

一个功能强大、完全去中心化的长截图拼接 Web 应用，基于 **FFmpeg.wasm** 实现。

## ✨ 核心特性
- **纯客户端计算**：视频无需上传，保证 100% 隐私安全。所有处理均在浏览器本地完成。
- **零成本运营**：可作为纯静态文件免费托管在 GitHub Pages 等平台。
- **全平台兼容**：现已完成针对 **桌面端** 和 **移动端** (iOS/Android) 浏览器的完美适配。

## 🚀 快速开始

### 环境需求
- Node.js (v18+)

### 安装步骤
```bash
# 在 standalone-web 目录下运行：
npm install
```

### 本地运行
```bash
npm run dev
```

### 一键部署至 GitHub Pages
```bash
npm run deploy
```
