# LooongImg Backend Core 🚀

[English] | [中文](./README_CN.md)

This is the original full-stack version of LooongImg, featuring a high-performance Node.js backend and a React frontend.

## 📂 Structure
- `/server`: Node.js Express server with FFmpeg and Canvas logic.
- `/client`: React frontend designed to communicate with the server API.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- FFmpeg installed and available in your system PATH.

### Installation

1. Install server dependencies:
```bash
cd server
npm install
```

2. Install client dependencies:
```bash
cd client
npm install
```

### Running Locally

From the `backend-core` root, you can start both services separately:

**1. Start Server:**
```bash
cd server
npm run dev:backend
```
*(The server usually starts on http://localhost:3001)*

**2. Start Client:**
```bash
cd client
npm run dev
```
*(Access the app at http://localhost:3000)*
